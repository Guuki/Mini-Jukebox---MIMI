# Mini Jukebox – Projeto Arduino
Este repositório contém o código e a documentação do projeto Mini Jukebox, desenvolvido como parte do curso de Bacharelado em Ciência da Computação – Instituto Federal de São Paulo (IFSP).
A proposta consiste em criar uma jukebox em miniatura totalmente funcional, capaz de detectar moedas, liberar um número correspondente de músicas, permitir navegação por botões, exibir imagens no display e reproduzir faixas armazenadas em um cartão SD.



## Funcionalidades
- **Leitura de moedas/fichas** através de TCRT5000
- **Sistema de créditos**: cada moeda libera uma música
- **Navegação por botões** para escolher faixas
- **Display TFT** exibindo imagem da música selecionada
- **Reprodução de áudio** via DFPlayer Mini
- Retorno automático para o menu inicial após término da música



## Componentes Utilizados
- Arduino Uno  
- Sensor infravermelho (TCRT5000)
- Botões  
- Display TFT  
- DFPlayer Mini  
- Cartão SD
